# Quantum Secure CVD AI Model 

# Overview
This project predicts cardiovascular disease using a quantum-inspired machine learning approach.

# Methodology
- Quantum feature mapping (sin, cos, x²)
- MobileNetV2 for image feature extraction
- Random Forest classifier
- Explainable AI (SHAP & LIME)

# Results
- Accuracy: 94.26%
- Precision: 0.96
- Recall: 0.91
- F1 Score: 0.93

#How to Run
1. Open the notebook in Google Colab
2. Upload dataset
3. Run all cells
